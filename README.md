# Station: A Learning Management Platform

Station is an innovative online learning platform designed for educators and students to create and join virtual classrooms. This system provides a streamlined experience for classroom management, tailored for modern teaching needs.

---

## Features

- **User Roles**:
  - **Teachers**: Create classrooms, set student limits, and manage participants.
  - **Students**: Join classrooms and interact in a structured learning environment.
- **Secure Login and Signup**:
  - Separate accounts for teachers and students.
  - Session-based authentication for secure user sessions.
- **Classroom Management**:
  - Teachers can create classrooms with custom names and student limits.
  - Students can join classrooms if slots are available.
- **Dynamic Components**:
  - Shared Navbar and Footer to maintain design consistency across all pages.
  - Bootstrap-powered responsive interface.
